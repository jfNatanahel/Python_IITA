# 🎓 Gestor Escolar en Python

Este proyecto es una aplicación de consola en Python que permite gestionar la información de alumnos en una escuela, incluyendo la **persistencia de los datos** mediante archivos `.json`.

## 📌 Funcionalidades

✔️ Llevar un registro de:
- Nombre, apellido, fecha de nacimiento y DNI del alumno  
- Nombre y apellido del tutor
- Registro de notas
- Cantidad de faltas
- Cantidad de amonestaciones

✔️ Mostrar datos de todos los alumnos  
✔️ Mostrar datos de un alumno específico  
✔️ Modificar datos (individuales, notas, faltas, etc.)  
✔️ Agregar nuevos alumnos  
✔️ Expulsar alumnos del sistema  
✔️ Guardar y cargar los datos automáticamente desde un archivo `datos.json`

## 🛠 Requisitos

- Python 3.x

## ▶️ Cómo ejecutar

1. Cloná el repositorio o descargá el archivo `gestor_escuela.py`
2. Abrí una terminal y ejecutá:
